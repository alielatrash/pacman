# Pacman Web Game

A classic Pacman game built with HTML, CSS, and JavaScript that runs in any modern web browser.

## Features

- Classic Pacman gameplay with maze navigation
- 4 colorful ghosts with AI (Blinky, Pinky, Inky, and Clyde)
- Power pellets that let you eat ghosts
- Score tracking and lives system
- Level progression
- Pause/Resume functionality
- Responsive design

## How to Play

1. Open `index.html` in your web browser
2. Click "Start Game" to begin
3. Use arrow keys to move Pacman:
   - ↑ Up Arrow - Move up
   - ↓ Down Arrow - Move down
   - ← Left Arrow - Move left
   - → Right Arrow - Move right

## Game Rules

- Eat all pellets to complete the level
- Avoid ghosts or you'll lose a life
- Eat power pellets (large yellow dots) to turn ghosts blue and eat them for bonus points
- You start with 3 lives
- Game ends when you run out of lives

## Scoring

- Small pellet: 10 points
- Power pellet: 50 points
- Eating a scared ghost: 200 points
- Completing a level: 500 bonus points

## Running the Game

Simply open `index.html` in any modern web browser. No server or build process required!

## Technologies Used

- HTML5
- CSS3 (with animations and gradients)
- Vanilla JavaScript (no frameworks)
